Instructions

  1. To join as an author, sign up using the provided link before the signup deadline. You may specify there whether you wish to be a solo author or if you want to work with co-authors.
  2. Within a few days of the deadline (hopefully), Evan will assign you an answer. You (and co-authors) need to write a puzzle that gives that specific answer. A deadline for a "first draft" of the puzzle is given, and of course you need to have a final draft before the hunt begins.
  3. Puzzles should be submitted in PDF format using the OPAL LaTeX template.
  4. Please talk to Evan as much as possible during the writing process. The whole point of this event is so that you can get feedback on the puzzle-writing process and learn from these comments.
  5. Solutions can be submitted in any format that's convenient to you. In practice, "link to Google doc" seems to be the easiest.
  6. Aim for easier puzzles, especially if you're writing for the first time. OPAL is meant to be a beginner hunt. Mystery-Hunt level puzzles are probably not going to be solved well.

Advice

  • If you're new to puzzle hunts, you should look through (or better yet, try to solve!) some past puzzle hunt puzzles to learn what you're signing up for. There's a couple beginner hunts listed on Evan's webpage.
  • Required reading:
    • David Wilson's puzzle-writing advice (2014)
    • Evan's puzzle-writing advice (2021)
  • Talk to Evan as much as you possible. Puzzle-writing is notoriously hard for beginners, and you'll be better off getting as much advice as you can on your specific drafts and ideas.
