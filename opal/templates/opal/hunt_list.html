About

The OTIS Puzzle Assembly Line, or OPAL, is a puzzle-potluck type event designed to help OTIS students get experience with writing and solving puzzlehunt-style puzzles. It's meant for beginners with little to no previous hunt experience.

So what is a puzzle hunt anyways?

Read Brian Chen's introduction! Evan also has some stuff written on https://web.evanchen.cc/puzzlehunt.html.

In short, a puzzle gives you some data, and you need to get an answer out of this data. Instructions are usually not provided, and figuring out how to interpret the data is part of the puzzle.

How does OPAL work?

In OPAL, any OTIS student can participate as a solver in the event.

In addition, each student (potentially with co-authors) can optionally submit up to one puzzle, hence the "potluck" description. (So you essentially get one free answer.) The deadlines to sign up to contribute a puzzle are shown above.

Each OPAL event consists of several feeder puzzles written by students, plus one meta-puzzle set by the game master that uses the answers to each of the feeders. The answer to the meta-puzzle will be a 24-26 digit hex code.
